# Raster style

    ellipsis.path.raster.style.add()

Adds a raster style.

**Mandatory arguments**
- pathId (uuid),
- name (string),
- method (string),
- parameters (object), containing
- token (uuid), your token

- description (string),

    ellipsis.path.raster.style.delete()

**Mandatory arguments**

- pathId (uuid),

- token (uuid), your token
